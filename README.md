## 프로젝트 소개

<p align="justify">
 책 평가 및 코멘트 서비스 웹 "Tree Book" 은 여러 독자들의 다양한 리뷰를 통해 책에 대한 의견을 공유하고,<br>
  구매 전에는 이미 읽어본 사용자들의 평가를 보고 결정할 수 있는 플랫폼입니다.
</p>
